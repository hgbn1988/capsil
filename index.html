Colonoscopia e Endoscopia Tradicionais

Problema fatal: Receptores custam milhares e devem ser compartilhados entre pacientes. Hospitais precisam comprar, higienizar e gerenciar equipamentos. Impossível vender em farmácia ou escalar para atenção primária.

Disponível • Limitado

Exames de Sangue Oculto

Teste simples e barato, mas com alta taxa de falsos negativos (40%). Detecta apenas sangramentos ativos, perdendo lesões pré-cancerosas. Importante como triagem inicial, mas insuficiente como prevenção efetiva.

⚠️ Importante: A CapSil é uma ferramenta de triagem. Diagnósticos definitivos sempre requerem confirmação por biópsia e/ou exames complementares quando indicado.

Detecção Visual Direta

  • ✓ Pólipos (>5mm: 92-96% sensibilidade)
  • ✓ Tumores visíveis
  • ✓ Úlceras ativas
  • ✓ Sangramentos ativos
  • ✓ Angiodisplasias

Alterações Sugestivas

  • → Padrões inflamatórios (DII)
  • → Atrofia vilositária (celíaca)*
  • → Alterações mucosas (Barrett)
  • → Lesões suspeitas
  • → Estenoses

*Requer biópsia + sorologia para confirmação

Não Detectável

  • ✗ H. pylori (precisa biópsia)
  • ✗ Grau histológico
  • ✗ Colite microscópica
  • ✗ Pólipos <3mm confiável
  • ✗ Infiltrado celular
